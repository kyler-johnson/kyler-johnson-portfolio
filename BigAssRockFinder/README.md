## Big Ass Rock Finder (B.A.R.F.)

<p align = "center">
  <img src = "B.A.R.F Logo.png" width = "500">
</p>

The B.A.R.F. is a scintillation detector that utilizes a Hamamatsu R1513 photomultiplier tube, five-inch organic scintillator, and zinc-sulfide screen. It is specifically designed to quickly detect radioactive mineral specimens in large low-background environments.
